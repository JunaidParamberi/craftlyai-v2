Dashboard — plan awareness

Free user, dashboard. How do they see their plan status without it feeling like an ad?

A

Usage card in the stats row

A 4th KPI card beside Revenue / Projects / Messages. Shows "3/3 clients · 4/5 docs" with small progress bars and an "Upgrade" link. Feels native to the dashboard, not an add-on.

B

Contextual banner — only when near a limit

No persistent widget. A slim top-of-page banner appears only when usage ≥ 80% of a limit. Disappears once dismissed. "You've used 4/5 docs this month — upgrade for unlimited." Silent otherwise.

C

No dashboard widget — avatar dropdown is enough

Dashboard stays clean. Plan info lives only in the avatar dropdown and at friction points (ghost rows, locked buttons). No proactive nudge on the dashboard at all.
