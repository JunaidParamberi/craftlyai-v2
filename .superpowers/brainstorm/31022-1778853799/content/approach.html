Plan-awareness across CraftlyAI

How should users discover their plan limits and feel the pull to upgrade?

A

Ambient Indicators Only

Plan badge in avatar dropdown. Quiet usage meters in sidebar footer (e.g. "3/3 clients"). No banners, no modals. Upgrade gate only appears when you hit a limit.

Pros

  • Never feels desperate
  • Zero clutter
  • Respects the user

Cons

  • Low discovery — most users won't notice
  • Weak conversion signal
B

Contextual Gate + Smart Nudges Recommended

Plan badge in avatar dropdown with usage summary. Inline limit chips at the exact moment they're relevant (e.g. client list shows "3/3 — Upgrade to add more"). Dashboard has one tasteful "unlock" card below the fold. No banners. Gate shows a smooth upgrade modal, not a redirect.

Pros

  • Feels like a feature, not an ad
  • Right time, right place
  • Clear value proposition at point of friction

Cons

  • More surface area to build
C

Dashboard-First Usage Dashboard

Dedicated "Plan Usage" section on the dashboard — big progress bars for clients, docs, AI actions. Avatar dropdown shows plan. No inline chips. Upgrade CTA lives only in the usage widget and billing page.

Pros

  • One clear place to see all limits
  • Not scattered

Cons

  • Users have to go to dashboard to see limits
  • Less contextual impact
